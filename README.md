# CodeFlow Mobile

Mobile-first IDE with built-in AI Agent. Rebuild of VS Code for mobile with cloud sandbox capabilities.

## Tech Stack

- **TypeScript + Vite** - Frontend development speed
- **Rust** - Editor core (syntax, LSP, Git)
- **Swift** - iOS native UI
- **Vite HMR** - 60fps React updates during dev
- **Alibaba OpenSandbox** - Cloud execution environment

## Features

### Mobile-First IDE
- Touch-friendly code editor
- Mobile file browser
- Voice input for code
- Camera scanning for code snippets
- Offline-first with sync

### Editor
- Syntax highlighting (100+ languages)
- IntelliSense autocomplete
- Git integration
- Multi-file editing
- Terminal emulator

### AI Agent
- Inline code completion
- Natural language commands
- Bug detection and fixes
- Code explanation
- Test generation
- Auto-debugging

### Cloud
- Remote code execution
- Containerized environments
- Team collaboration
- Deployment pipelines

## Getting Started

```bash
# Clone the repo
git clone https://github.com/admint11/codeflow-mobile.git

# Install dependencies
npm install

# Start development
npm run dev
```
